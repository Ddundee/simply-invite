# Simply Invite

InviteLink is a event invitation app where users can upload their own invitation designs, generate a shareable link, and choose to send invitations via email or SMS.

- Posthog & Sentry (for analytics)
- Drizzle (for orm)
- Supabase (for db)
- Vercel (for hosting)
